# 42. Trapping Rain Water

**Hard**

Dado um array de inteiros não negativos representando um mapa de elevação (onde a largura de cada barra é 1), calcule quanto de água pode ser armazenado após a chuva.

---

| Exemplo   | Entrada                              | Saída | Explicação                                                          |
| --------- | ------------------------------------ | ----- | ------------------------------------------------------------------- |
| Exemplo 1 | `height = [0,1,0,2,1,0,1,3,2,1,2,1]` | `6`   | A água fica presa entre as elevações, resultando em **6 unidades**. |
| Exemplo 2 | `height = [4,2,0,3,2,5]`             | `9`   | Aqui, vales entre os blocos acumulam **9 unidades** de água.        |

---

### Restrições

- `n == height.length`
- `1 <= n <= 2 * 10^4`
- `0 <= height[i] <= 10^5`
