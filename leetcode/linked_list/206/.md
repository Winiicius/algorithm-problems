# Reverse Linked List

Given the head of a singly linked list, reverse the list, and return the reversed list.

---

| Example   | Input                | Output        |
| --------- | -------------------- | ------------- |
| Example 1 | `head = [1,2,3,4,5]` | `[5,4,3,2,1]` |
| Example 2 | `head = [1,2]`       | `[2,1]`       |
| Example 3 | `head = []`          | `[]`          |
